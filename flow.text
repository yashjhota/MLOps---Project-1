3. Create a virtual env, activate it and install the requirements from requirements.txt
   conda env list (TO check all the created enviroment)
   conda env remove -n <name> (TO delete)
   conda create -n vehicle python=3.10 -y  (To create a new virtual enviroment)
   conda activate vehicle
   (pip list to check all )
     Do "pip install -r requirements.txt"    (add required modules to requirements.txt)
   
4. Do a "pip list" on terminal to make sure you have local packages installed.
